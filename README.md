# TrackadenZ 🏋️

Dein persönlicher Fitness- & Ernährungs-Tracker.

## Kostenlos online stellen – Schritt für Schritt

### Was du brauchst
- Einen GitHub Account (github.com) – kostenlos
- Einen Netlify Account (netlify.com) – kostenlos
- Diesen Ordner als ZIP-Datei

---

### Schritt 1 – GitHub Repository erstellen
1. Geh auf github.com → einloggen
2. Klick auf das **+** oben rechts → „New repository"
3. Name: `trackadenz`
4. Auf **„Create repository"** klicken

### Schritt 2 – Dateien hochladen
1. Im leeren Repository auf **„uploading an existing file"** klicken
2. Alle Dateien aus diesem Ordner (entpackt) **per Drag & Drop** ins Fenster ziehen
3. Wichtig: Auch den Unterordner `src/` mit hochladen
4. Auf **„Commit changes"** klicken

### Schritt 3 – Netlify verbinden
1. Geh auf netlify.com → einloggen / Account erstellen
2. Klick auf **„Add new site"** → **„Import an existing project"**
3. Wähle **„GitHub"** → dein Repository `trackadenz` auswählen
4. Build-Einstellungen werden automatisch aus `netlify.toml` gelesen
5. Auf **„Deploy site"** klicken

### Schritt 4 – Fertig! 🎉
Nach ca. 1–2 Minuten bekommst du eine URL wie:
`https://trackadenz-abc123.netlify.app`

Diese URL kannst du mit jedem teilen!

### Optional: Eigene Domain / URL
In Netlify unter „Domain settings" → „Add custom domain" kannst du einen eigenen Namen vergeben, z.B. `mein-trackadenz.netlify.app` (kostenlos).

---

## Auf dem Handy installieren (als App)

### iPhone (Safari)
1. Link in Safari öffnen
2. Teilen-Button (Quadrat mit Pfeil) tippen
3. **„Zum Home-Bildschirm"** → „Hinzufügen"

### Android (Chrome)
1. Link in Chrome öffnen
2. Drei-Punkte-Menü → **„Zum Startbildschirm hinzufügen"**

Die App erscheint dann wie eine normale App auf dem Homescreen – ohne App Store!
